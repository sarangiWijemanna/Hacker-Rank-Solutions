## Task

Query all columns for a city in ```CITY``` with the ID 1661.

The ```CITY``` table is described as follows:

<img align="center" src="/Images/Select By ID.png" alt="icon"/>

### Expected Output

```
1661 Sayama JPN Saitama 162472
```

## Solution

```
SELECT *
    FROM CITY
    WHERE ID = 1661;
```
